# 🚀 Pallavi AI Suite – NLP & Generative AI Project

An end-to-end **AI-powered application** built using Hugging Face ecosystem that combines **Natural Language Processing (NLP)** and **Generative AI** capabilities into a single interactive platform.

This project demonstrates real-world implementation of multiple AI tasks including **Text Generation, Named Entity Recognition (NER), Sentiment Analysis, and Text-to-Image Generation** with an intuitive UI.

---

## ✨ Key Highlights

- 🧠 Multi-task AI system (NLP + Generative AI)
- 🤗 Powered by Hugging Face Transformers & Diffusers
- 🎨 Realistic image generation using Stable Diffusion
- 🌐 Interactive web UI using Gradio
- ⚡ Beginner-friendly yet production-ready structure

---

## 🔥 Features

### ✍️ 1. Text Generation (GPT-2)
Generate human-like text responses from user input.

### 🧠 2. Named Entity Recognition (NER)
Identify entities like **Person, Location, Organization** from text.

### 😊 3. Sentiment Analysis
Classify text as **Positive / Negative** using pretrained models.

### 🎨 4. Text-to-Image Generation
Generate images from text prompts using **Stable Diffusion model**.

### 🌐 5. Interactive UI
Simple and clean interface using **Gradio**.
